About the event

The Appwrite Hackathon on DEV was announced on April 12th and consisted of building an application under one set of categories. One of those categories was Mobile apps, and in order to participate, I created an app using their SDK and I created a post on the Dev.to where it should include a description of the project, public repository and evidence of how it works.

My participation

I participate in creating an Android App. It consists of a Puzzle Game called Nigma, which uses technologies like Jetpack Compose, coroutines, and the AppWrite SDK. The app was born from the idea of having a simple game with an interface similar to the new material 3.
